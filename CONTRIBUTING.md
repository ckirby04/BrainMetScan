# Contributing to BrainMetScan

This project is actively seeking collaboration, especially in the following areas:

## What We're Looking For

- **Multi-site MRI datasets** for external validation and generalizability testing
- **Clinical partners** interested in treatment planning applications
- **Researchers** working on small lesion detection or brain metastasis segmentation
- **Radiologists** willing to provide expert annotations for model evaluation

## How to Contribute

1. **Open an issue** describing what you'd like to work on
2. **Fork the repo** and create a feature branch
3. **Submit a pull request** with a clear description of changes

## Development Setup

```bash
git clone https://github.com/ckirby04/BrainMetScan.git
cd BrainMetScan
pip install -r requirements.txt
```

See [`data/README.md`](data/README.md) for dataset setup instructions.
